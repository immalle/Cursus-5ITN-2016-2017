# CSharp Interactive

In Visual Studio: `View -> Other Windows -> C# Interactive` (zet best full screen).

Raadpleeg `#help`. Vooral `ALT+UP` en `ALT+DOWN`, alsook `HOME` en `END` zijn handig.

> De cursor moet achteraan staan om een commando uit te voeren (`END`).
> Of: `CTRL+ENTER`.



## Onderwerpen die zich goed lenen om op deze manier te oefenen

Uiteraard kan je op deze manier geen GUI gebruiken
maar het is wel een snelle manier om uit te zoeken
hoe bepaalde classes en andere C#-mechanismes werken.
Het is altijd belangrijk om de return-waarde van uitgevoerde
methods te bestuderen. Ook foutmeldingen zijn zeer leerrijk.

- :hash: `4` : Declareren en initialiseren van variabelen, casten naar andere types, format strings, ...
- :hash: `5` : (Kleine) methods definiëren en aanroepen, b.v. `AreaRectangle`
- :hash: `6.7` : De class `Random` uitproberen, `Random`-objecten maken, ...
- :hash: `13.2` : Generieke lijsten aanmaken en erover itereren b.v. `List<int>`, `List<string>`, ...
- :hash: `16` : String-bewerkingen, b.v. `ToUpper`, `LastIndexOf`, ...
- :hash: `14` : Arrays (van int's, strings, ...)
- (:hash: `15` : Multidimensionale arrays)
- :hash: `18` : Bestanden lezen/schrijven, directories uitlezen, ... met de classes `System.IO.Directory` en `System.IO.File`
